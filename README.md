# JToolBox
## 简介：
免费的Minecraft工具箱，目前包含BanTracker
## 如何使用：
安装.net framework 4.7.2及以上<br>
## 兼容性：
Mica效果只支持win11<br>
目前仅测试过Windows 10 22H1的运行效果<br>
界面设计器不支持Costura.Fody
